Visual Identity & Constants

Documentation of the design system and telestration vocabulary.

Design Tokens (style.css)
Variable Value Usage
--primary #42a5f5 Brand color, active states, and primary graphics.
--secondary #9ea7af De-emphasized text, borders, and inactive states.
--bg #121212 Deep black background for high-contrast video viewing.
--card-bg #1e1e1e Surface color for controls, navigation, and cards.
--text #e0e0e0 Primary readable text color.
--danger #ef5350 Errors, delete actions, and opponent team highlights.
--success #66bb6a Confirmations, completed states, and home team highlights.
Telestration Shapes

Tracking Halo

An elliptical ring used to highlight players. Features a sinusoidal pulse.

.ellipse-element {
  border-radius: 50%;
  background: rgba(66, 165, 245, 0.2);
}

Selected State

Active shapes feature a white border and secondary primary-color glow.

.selected {
  border-color: #fff;
  box-shadow: 0 0 20px var(--primary);
}

Tactical Arrow

Vector paths for movement. Built with SVG paths and arrowheads.

.curve-element {
  stroke: var(--primary);
  stroke-width: 4;
}
Global Layout Constants

The system uses dynamic eye-based scaling for stereoscopic top-bottom and side-by-side viewing.

--eye-width: 45vw;
--eye-height: calc(var(--eye-width) * 9 / 16);
